Summary

This project was inspired originally by Kenny Chenâ€™s original black hole project from sp20. We originally wanted to do a full on expansion of that project, but after learning about the flexibility of particle simulation (after looking at the game Noita) we figured we could do something more general by recreating scenes using a set of predefined particles that simplify the simulation. We wish to animate interesting scenes and have a random element added to our particles that make them more lively. Depending on the textures given, we should be able to create and animate scenes based on how our particles interact realistically with light, with each other, and with other particles.

Purpose

The purpose of this project is to use particle simulation to make animated, interesting, 2-dimensional scenes. This is not an interactive demo, rather a scene with predefined initial conditionals that we render for 10 seconds.
